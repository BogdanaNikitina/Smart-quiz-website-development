#--------------------------------------------------------------Проект smartquiz.ru--------------------------------------------------------------------------------------------#

#----------------------------------------------------------------Реализованный функционал проекта:----------------------------------------------------------------------------#
1. Реализован веб-сайт с несколькими вкладками и смарт квизом. 
Вкладки на сайте:
"Главная" - главная и основная страница сайта, на коорой написана краткая информация и сведенья о опросе.
"О нас" - на этой вкладке указаны разработчики сайта и их контактные данные (ссылки на тг)
На смарт-квиз можно перейти после нажатия на кнопку "Пройти опрос"
Смарт-квиз сосоит из 6 вопросов, после ответа на которые, данные отправляются в БД.

#----------------------------------------------------------------Основной стек технологий проекта:----------------------------------------------------------------------------#

CSS
JS 
PHP

#-----------------------------------------------------------------Инструкция по запуску:--------------------------------------------------------------------------------------#
1. Скачайте файлы проекта и установите OSPanel со всеми зависимостями. 
2. Папку smarttquiz.ru поместите в C:OSPanel\home\ 
3. После этого запустите OSPanel  и зайдите в PhpMyAdmin
4. Вход в phpMyAdmin. По умолчанию в Open Server: Логин: root Пароль: оставьте пустым
5. В верхнем меню выберите вкладку "SQL" и вставьте следующий код:
CREATE DATABASE IF NOT EXISTS `quiz_design`
CHARACTER SET utf8mb4
COLLATE utf8mb4_unicode_ci;
Нажмите кнопку "Вперед"
6. Откройте SQL-редактор. После выбора базы данных quiz_design в левой колонке в верхнем меню нажмите вкладку "SQL" и выполните SQL-запрос для создания таблицы
(просто скопируйте и вставьте этот код в поле SQL-запроса):
CREATE TABLE IF NOT EXISTS `quiz_results` (
    `id` INT(11) NOT NULL AUTO_INCREMENT,
    `room_type` VARCHAR(100) DEFAULT NULL,
    `zones` TEXT DEFAULT NULL,
    `area` INT(11) DEFAULT NULL,
    `style` VARCHAR(100) DEFAULT NULL,
    `budget` VARCHAR(100) DEFAULT NULL,
    `name` VARCHAR(255) DEFAULT NULL,
    `phone` VARCHAR(50) NOT NULL,
    `email` VARCHAR(255) DEFAULT NULL,
    `comment` TEXT DEFAULT NULL,
    `page_url` TEXT DEFAULT NULL,
    `utm_source` VARCHAR(255) DEFAULT NULL,
    `ip_address` VARCHAR(45) DEFAULT NULL,
    `user_agent` TEXT DEFAULT NULL,
    `created_at` TIMESTAMP NOT NULL DEFAULT CURRENT_TIMESTAMP,
    PRIMARY KEY (`id`),
    INDEX `idx_phone` (`phone`),
    INDEX `idx_created` (`created_at`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;
После этого нажмите кнопку "Вперед" (Execute) внизу страницы и должно появиться сообщение: "Запрос выполнен успешно"
7. Теперь через OSPanel запустите -smartquiz.ru
